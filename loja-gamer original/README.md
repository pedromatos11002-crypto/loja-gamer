# Loja Gamer

Projeto de uma loja virtual gamer desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades
- Página inicial
- Página de produtos
- Carrinho de compras
- Tela de login
- Layout gamer com imagens
- Armazenamento com localStorage

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- VS Code
- GitHub
